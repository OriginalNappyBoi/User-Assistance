Download the Desktop app for Trello

A step by step list of how to download the desktop app for trello.

We should probably do both versions for this task, IE do a set of instrucitons for PC and one for Mac users.

You will need this list of materials inorder to download the PC version of Trello.

Materials:
  • Working Laptop/ desktop
  • Windows OS 10 or higher
  • Microsoft store account
  1. Inorder to download the Trello desktop app for windows you must have a microsoft store account. See here on how to create a microsoft store account
  2. Once you have an account you can search for Trello or in web browser search Trello desktop app.
  3. Find the version you need, for these instructions we will be following the windows parameters, once you have found the version you need click to install
  4. If you are searching from within a web broswer you will now be prompted to sign into the microsoft store, if you are already in the microsofrt store it will begin downloading.
  5. After it is finished downloading, the appplicaion will run, click continue, then either select login or signup.
  6. Once you have done that you are now ready to use Trello as an application and no longer have to use it in a web browser.
