Adding a list

A list is a collection of cards that represent a specific topic.

Perform the following steps to add a list to your board:

  1. In the middle of the screen you have an option to add lists to your already created board. If you wish to add lists to your board, click the "add list" option on the screen. see Hey(Figure 1).
  2. After clicking this option you will need to enter a title for the list you want to add. Once you have added the tittle click add (Figure 2).
  3. You have now created a list within your board, and now you can add different cards, lists, and other things to it (Figure 3).
