Types of Trello Accounts

Be sure you sign up for the Trello account that is best for you.

Pricing

There are three different types of Trello accounts:
  • Free ($0 forever)
  • Business Class ($9.99 per user per month anually)
  • Enterprise ($20.83 per user per month for 20 users)

Standard Features

The table below shows the standard features associated with the different types of Trello accounts.

Free Business Class Enterprise
Unlimited personal boards Unlimited personal boards Unlimited personal boards
Unlimited cards Unlimited cards Unlimited cards
Unlimited lists Unlimited lists Unlimited lists
10MB per file attachment 250MB per file attachment 250MB per file attachment
- Priority support Priority support
- Observers Observers
- Custom backgrounds and stickers Custom backgrounds and stickers

For more information about the feature differences in Trello accounts, see https://trello.com/en-US/pricing.
