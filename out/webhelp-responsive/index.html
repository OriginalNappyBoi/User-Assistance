Printing and Exporting
Trello’s Print and Export feature is available to board creators and collaborators. To access this feature, natigate to Board Menu > More > Print and Export. The Export feature allows boards to be exported in JSON format, which uses text to store and exchange data. The Print feature produces a printable version of your board including all lists and cards.
